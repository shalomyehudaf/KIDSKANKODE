A variable is a name that you create and then use it in your program instead of the actual thing you want to use in your program. For example: Let's say I have a variable named vary1, and I want vary1 to represent a string("This is a string of text"), integer(6), boolean(True/False), or float(4.65). So I will type a line that looks something like this. vary1 = "This string is represented by vary1" Any time I refer to vary1 in my program, the computer will automatically replace vary1 with "This string is represented by vary1". For example, if I write a line of code along the lines of 'the text that should be displayed on the screen is vary1', then when the computer reads it, it will read the line of code differently: 'the text that should be displayed is "This string is represented by vary1". (When we get up to the lesson about print statements, you will see the actual code looks very different than 'the text that should be displayed on the screen is vary1', but we'll worry about that later.) A quick rundown on what all of the different variable types are: string - a sentence or word(s) grouped together by " " or ' ' integer - a number without a decimal point(.), can be positive or negative float - a number with a decimal point (which can be 0.0), can be positive or negative boolean - either True or False, with 'and', 'or', and 'not' but more on that later YOU TRY IT!
