The difference between coding and programming is something that most people do not understand.
Programming is the ability to think about and understand code. Coding is actually putting that programming into action.
In Scratch you do that by using drag and drop blocks (for more about Scratch go to The Scratch Website").
In HTML you use tags to set up the webpage and add things to it.
JavaScript is text-based and uses a lot more punctuation marks, such as: semi-colon(;), quotes("_" '_'), colon(:), comma(,), and period(.).
It also uses several types of brackets, such as: (, [, {, }, ], ) and slashes(\/).
In Python, you can use functions, variables, and classes to make amazing things happen.
